미세조정 사용법
1) 미세조정 모드 ON → 원/박스를 마우스로 드래그
2) 스냅 ON이면 5px 단위로 “착” 붙어서 정렬이 쉬움(OFF도 가능)
3) 끝나면 좌표 내보내기로 좌표를 복사해두면 안전(백업)
※ 드래그로 움직이면 자동 저장되므로 새로고침해도 유지됨
Raw 데이터 붙여넣기 (엑셀 범위 복사 → 여기 클릭 → Ctrl+V)
- 신규 포맷: TANK NO.(B열) / 측정량(D열) / 제품명(F열)
- 도면에 없는 탱크는 무시 / Br1~Br12는 이름만 표시 / 말분 무시
